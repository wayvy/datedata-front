# DateData Core App
Main application for DateData.

## App Structure
```
src/
├── components/ # UI components
├── config/ # Configuration
├── features/ # Features
├── hoc/ # Higher-order components
├── layouts/ # Layouts
├── pages/ # Pages
├── stores/ # MobX stores
├── styles/ # Global styles
├── widgets/ # Widgets
└── main.tsx # Entry point
```

## Tech Stack
- React
- TypeScript
- MobX
- SCSS
